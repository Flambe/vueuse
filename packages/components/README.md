# @velocity1/vueuse-components

[![NPM version](https://img.shields.io/npm/v/@velocity1/vueuse-components?color=a1b858)](https://www.npmjs.com/package/@velocity1/vueuse-components)

> This is an extension of [VueUse](https://github.com/vueuse/vueuse), enables renderless component usage for composable functions.

## Install

```bash
npm i @velocity1/vueuse-components @velocity1/vueuse-core
```

Learn more about [component usage](https://vueuse.org/guide/components).
